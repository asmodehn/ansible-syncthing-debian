# Ansible Syncthing Debian

Provides Syncthing Debian packages from <http://apt.syncthing.net/>.

## Configurations

Default:

~~~ yaml
syncthing_version: "0.12.10"
syncthing_pin_priority: 600
syncthing_use_ufw: false
~~~
